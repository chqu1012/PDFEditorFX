# PDFEditorFX
A small javafx pdf editor supports

- [x] Pdf Search
- [x] Pdf to text
- [x] Navigate pdf via Outline

Future plan:
- [ ] Create bookmarks
- [ ] Export to documents
- [ ] Export to html
- [ ] Export to image
- [ ] Render latex files
- [ ] Render restructured files
- [ ] Render markdown files
- [ ] Presentation mode
- [ ] Drawing shapes in pdf
- [ ] Highlighting pdfs
