# PDFEditorFX
A small javafx pdf editor supports

- [/] Pdf Search
- [/] Pdf to text
- [/] Navigate pdf via Outline
- [x] Create bookmarks
- [x] Export to documents
- [x] Export to html
- [x] Export to image
- [x] Render latex files
- [x] Render restructured files
- [x] Presentation mode
- [x] Drawing shapes in pdf
- [x] Highlighting pdfs
